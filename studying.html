Python Interview Questions Study Guide

Topics

What is Python?

Python is a high-level, interpreted programming language known for its readability and versatility.

Key features of Python:

  • Interpreted and dynamically typed
  • Object-oriented
  • Extensive standard library
  • Platform-independent

Mutable vs Immutable:

  • Mutable: List, dict, set
  • Immutable: int, str, tuple

Control Structures:

  • Conditional Statements: if, elif, else
  • Loops: for, while, break, continue

Functions:

Defined using the def keyword. Example:

def greet(name):
    return f"Hello, {name}!"

Lambda functions are anonymous functions created using lambda.

What is OOP?

Object-Oriented Programming is a paradigm based on the concept of objects that contain data and methods.

Example:

class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        return f"{self.name} says Woof!"

dog = Dog("Buddy")
print(dog.bark())

Iterators and Generators:

Iterators are objects that can be iterated upon, and generators are a simplified way to create iterators using yield.

Example of Generator:

def count_up_to(n):
    count = 1
    while count <= n:
        yield count
        count += 1

for num in count_up_to(5):
    print(num)
